The Sorting Visualizer is an interactive web-based application designed to help students and developers understand sorting algorithms visually.
It represents array elements as 3D cubes and animates comparisons, swaps, and placements in real time.

The project is built using HTML, CSS, JavaScript, and Three.js, focusing on clean UI, smooth animations, and algorithmic clarity.
Users can control animation speed, step through algorithms manually, and compare algorithm performance using statistics like comparisons and swaps.

This project demonstrates strong fundamentals of Data Structures & Algorithms, JavaScript, and 3D graphics programming.
